# smartlift
SmartLift is an intelligent safety system for logistics and warehousing. It generates a unique QR code for each item, encoding weight and center of gravity. By scanning the code, workers receive collaboration recommendations (e.g., "two-person lift required") and posture guidance. Integrated with AI vision, the system analyzes lifting postures in real-time and triggers voice alerts when detecting dangerous movements. This creates a complete safety loop from pre-operation information to in-action correction. Deployed at low cost using existing mobile devices, SmartLift transforms passive training into active intervention, effectively reducing the risk of musculoskeletal disorders.
